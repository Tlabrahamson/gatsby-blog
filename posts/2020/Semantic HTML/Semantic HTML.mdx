```jsx
  <div class="full-container">
    <div>
      <h1>A Real Good Heading</h1>
      <ul>
        <li><a href="#" target="_blank">Home</a></li>
        <li><a href="#" target="_blank">Products</a></li>
        <li><a href="#" target="_blank">About</a></li>
        <li><a href="#" target="_blank">Contact</a></li>
      </ul>
    </div>
    <div>
      <h3>&copy; 2020 Good Company</h3>
    </div>
  </div>
```
The HTML above is *not* semantic.

You can achieve Semantic bliss by using semantic HTML elements that describe their meaning to the browser and the user. Using semantic elements increases the readability of your HTML and improves the SEO (Search Engine Optimization).
